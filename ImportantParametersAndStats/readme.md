* Connect to the server: `PS D:\Programs\mongosh-1.8.2-win32-x64\mongosh-1.8.2-win32-x64\bin> .\mongosh.exe "mongodb://localhost:27017/?readPreference=primary&directConnection=true&ssl=false"`
